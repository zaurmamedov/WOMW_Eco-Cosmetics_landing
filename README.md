# WOMW Eco Cosmetics Landing

`WOMW Eco Cosmetics Landing` is a responsive promotional landing page for the "Witch of Mountain Winds" eco-cosmetics brand. The project presents the brand story, featured products, sustainability values, and contact information in a visually rich single-page layout.

## Live Preview

- [Demo link](https://zaurmamedov.github.io/WOMW_Eco-Cosmetics_landing/)

## Design Reference

- [Figma Design](https://www.figma.com/design/Fz588JKGuPS2Bk21De4KE5/Brand-of-eco-cosmetics-_FE-students?node-id=21779-2&p=f&t=NZ8Mbeql4TFK2XRf-0)

## Technologies Used

- HTML5
- SCSS (Sass)
- JavaScript
- Parcel
- ESLint
- Stylelint

## Features

- Responsive landing page layout for mobile, tablet, and desktop screens
- Product showcase section with category navigation
- Off-canvas mobile menu
- Brand storytelling and sustainability-focused content blocks
- Contact section with social media links

## Getting Started

Clone the repository:

```bash
git clone https://github.com/zaurmamedov/WOMW_Eco-Cosmetics_landing.git
cd WOMW_Eco-Cosmetics_landing
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm start
```

Build the project for production:

```bash
npm run build
```
